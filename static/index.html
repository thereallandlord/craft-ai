Инструкция по настройке n8n

Для каждого workflow в n8n нужно добавить Webhook ноду:

  1. Откройте workflow в n8n
  2. Добавьте ноду "Webhook" (HTTP Method: POST)
  3. Подключите её параллельно к существующему executeWorkflowTrigger
  4. Скопируйте Production URL вебхука и вставьте выше
